Aparte de los arreglos $_POST y $_GET, ¿qué otros arreglos están predefinidos en php y cuál es su función?

  • $GLOBALS: variables definidas en el global scop del script
  • $_SERVER: contiene los headers y direcciones.
  • $_FILES: Contiene los items descargados via el método de HTTP POST
  • $_COOKIE: Pasa el script actual via HTTP Cookies
  • $_SESSION: cientie las sesiones variables disponibles para un script
  • $_REQUEST: Tiene el contenido de $_GET, $_POST y $_COOKIE
  • $_ENV: Pasa un variable mediante el método environment

    ¿Qué es XSS y cómo se puede prevenir?

    Cross-site scripting es un tipo de vulnerabilidad de seguridad de aplicaciones web en la cual se permite inyectar código JavaScrpt u otro similar. Existen ciertas medidas para exitar XSS:

  • No incluir datos foráneos
  • Codifique y valide todos los datos y atributos a incluir en el html, Javascript y URLs
  • Genere taglis que se engarfuen de codificar los datos de salida
  • Usar librerias para controlar la codificación de algunos campos
