Preguntas

¿Qué es ODBC y para qé es útil?

Open Database Connectivity es una interfaz de programación de estándar abierto para acceder a una base de datos. Es útil ya que por emdio de sentencias te permite hacer consultas de los diferentes archivos dentro de una BD

¿Qué es SQL Injection

Es una vulnerabilidad que permite inyectar código mediante intrucciones de SQL de manera mal intencionada para manipular las bases de datos mientras pone en peligro la integridad de los archivos almacenados.

¿Qué técnicas puedes utilizar para evitar ataques de SQL Injection

Existen herramientas para hacer pruebas y detectar vulnerabilidades, como SQLMap, también se deben evitar las sentencias dinámicas, también existen diversas funciones que te permiten limpiar las entradas y evitar los ataques.
