    ¿Qué diferencias y semejanzas hay entre Java y JavaScript?
    Java es un lenguaje de programación, JavaScript no es un lenguaje compilado, sino interpretado. En Js no se declara el tipo de variables y sabe excribir en el disco duro externo.

    ¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)

  • Date.now() -> hora actual
  • Date.parse() -> devuelve el numero de milisegundos desde enero de 1970.
  • Date.UTC() -> devulve el numero de milisgundos en un objeto desde enero de 1970.
  • getDate() -> valor dell dia del mes usando la hora local
  • getYear() ->Devuelve el valor de año.


    • ¿Qué métodos tienen los arreglos? (Menciona al menos 5*)
  • Array.form() -> Cra un nueo Array de un objeto.
  • Array.isArray() -> Devuelve true si es un array.
  • Array.observe() -> Observa cambios del array asincronicamente
  • Array.of() -> crea un nuevo Array con un numero variable de argumentos.
  • Array.prorotype.shift() -> elimina el primer elemento de una matriz unidimensional.


    • ¿Cómo se declara una variable con alcance local dentro de una función?
    Con "var" o "let".

    ¿Qué implicaciones tiene utilizar variables globales dentro de funciones?
    Pueden confundirse con otras variables al haber coincidencias en el nombre o pueden cambiar a lo largo del código y ocasionar errores.

    Existen varios métodos como: evec, test, match, search, split, etc. Podría servir para encontrar coincidencias de algún nombre o dato en especifico que este vinculado con más datos.
