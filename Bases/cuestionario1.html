Cuestionario 1

¿Qué es entonces una base de datos - DB?
Es conjunto de información, de archivos interrelacionados, ciudadosamente diseñada para evitar la repetición. [1]
¿En qué casos es conveniente usar bases de datos?
En cualquier situación donde se amerite una estructura organizada de archivos interrelacionados y con un gran volumen de datos.
¿Qué es un sistema de gestión de base de datos - DBMS?
Consiste en un programa que te permita acceder a una colección de datos que se relacionan entre sí. Su principal objetivo es dar un ambiente eficaz para extraer y almacenar.

Enlista y explica las funciones/responsabilidades que tienen los DBMS.

  • Organizar la base de datos.
  • Manejar los datos basándose en los requerimientos del usuario.
  • Registrar el Uso de la Base de datos.
  • Interacción con el manejador de archivos.
  • Respaldo y Recuperación.
  • Contar los mecanistos de control de consistencia de los datos. [2]

Ejemplifica y justifica en que proyecto de los que has realizado hubiera sido conveniente utilizar una base de datos.
En el verano realicé unas muestras para un proyecto piloto de Análisis de Señales de EEG. Tenía la información regada en distintas plataformas y muchos datos los tenía escritos en una libreta. Una Base de datos hubiera hecho todo el manejo de la información mucho más fácil y eficiente.
